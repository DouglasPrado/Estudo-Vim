# Lista de Comandos VIM
*Modo Insert*
- i = Modo INSERT
- esc = Voltar para o modo normal
- :w = Salvar
- :q = Sair
- :wq = Salvar e sair

*Modo Normal*
Setas:
  - L = Direita
  - J = Baixo
  - k = Cima
  - H = Esquerda
  - Numero + [Letra das Setas] = Numero de vezes que a seta vai se repetir
  
# Arquivo de Configurações VIM
- vim .vimrc = Entra no arquivo
- colorscheme industry
- set tabstop=2
